# Python Calculator

A basic calculator built using Python.

This project was created while learning Python fundamentals and practicing problem-solving.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulo (%)
- Power (**)
- User input support
- Result output
- Basic conditional logic

## Technologies Used

- Python

## Concepts Practiced

- Variables
- int(input())
- Arithmetic operators
- if / elif / else statements
- Boolean logic
- Error handling basics

## How to Run

1. Install Python
2. Download or clone the project
3. Open terminal in the project folder
4. Run:

```bash
python calculator.py
```

## Example

```text
Enter first number: 10
Enter operator: +
Enter second number: 5

Result: 15
```
